# Rootling Website

This `docs/` folder contains the GitHub Pages website for Rootling.

## Pages

| Page | URL | Description |
|------|-----|-------------|
| Home | `/Rootling/` | App landing page with features and download CTA |
| Privacy | `/Rootling/privacy` | Full privacy policy |
| Support | `/Rootling/support` | Support contact (email only) |
| Help | `/Rootling/help` | Help guides, tutorials, and FAQ |

## Setup

1. Push this repo to GitHub at `pinkysworld/Rootling`
2. Go to **Settings → Pages**
3. Set **Source** to "Deploy from a branch"
4. Set **Branch** to `main` and folder to `/docs`
5. Save — your site will be live at `https://pinkysworld.github.io/Rootling/`
